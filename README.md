# AMOORY_STORE - متجر الألعاب والبطاقات الرقمية

موقع إلكتروني بسيط لمتجر بطاقات الألعاب والبطاقات الرقمية.

## كيفية الاستخدام

1. افتح ملف `index.html` في أي متصفح ويب
2. الموقع لا يحتاج إلى خادم - يمكن فتحه مباشرة من الملف

## الملفات

- `index.html` - الصفحة الرئيسية للموقع
- `styles.css` - ملف التنسيقات
- `script.js` - الكود الخاص بالتفاعلية
- `images/` - مجلد الصور

## الميزات

✅ تصميم عربي كامل مع دعم RTL
✅ واجهة عصرية وجذابة
✅ متجاوب مع جميع أحجام الشاشات
✅ سلة تسوق تفاعلية
✅ عرض المنتجات حسب الفئات
✅ قسم العروض الخاصة
✅ تقييمات العملاء
✅ نموذج النشرة البريدية

## التخصيص

يمكنك تعديل:
- المنتجات في ملف `script.js` (متغير `productsData`)
- الألوان في ملف `styles.css` (متغيرات CSS في `:root`)
- النصوص في ملف `index.html`
